# git-webhooks-tester

A CLI tester for Hygie rules.

## Goal

This tool allows you to send customize Git Events to your Hygie local app, for testing purpose.

## Getting Started

Just run `node src/index.js` and customize your Git events !

Events availables:

- Push,
- NewBranch,
- NewIssue,
- NewPR,
- IssueComment,
- PRComment.
